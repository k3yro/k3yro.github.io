## Requirements
- [.Net Framework 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472 "Microsofts's Homepage") (or higher)
- A folder full of files
## Regex Syntax
| Syntax   |  Description  |
|----------|:-------------:|
| ^        |  Begin of filename |
| $        |  End of filename   |
| \s       | Whitespace |
|[A-z]     | Find chars A-z |
